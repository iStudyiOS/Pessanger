# Pessanger
위치공유를 기반으로 주변에 있는 친구의 위치를 확인하고, 친구와 채팅을 할 수 있는 앱.

## 사용 API
- kakao 지도 API를 사용하였습니다. [API 링크](https://apis.map.kakao.com/)
